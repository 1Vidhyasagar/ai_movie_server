# 🎬 AI Movie Genius – Backend

Express server connected to Claude 3 via OpenRouter to fetch movie recommendations.

## 💡 Stack

- Node.js + Express
- OpenRouter Claude 3 API
- CORS + dotenv

## ⚙️ Run

```bash
npm install
node server.js
```
